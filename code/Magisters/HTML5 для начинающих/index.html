Веб-дизайн (от англ. Web design) — отрасль веб-разработки и разновидность дизайна,
в задачи которой входит проектирование пользовательских веб-интерфейсов для сайтов веб-приложений.
Веб-дизайнеры проектируют логическую структуру веб-страниц, продумывают наиболее удобные решения подачи
информации и занимаются художественным оформлением веб-проекта.
